🛠️ O Projeto: "SnapStorage"O objetivo é criar uma API onde você envia uma imagem, ela é salva em uma pasta local, e o caminho (path) e metadados são registrados no MySQL.📋 Cronograma Sugerido (90 min)TempoEtapaFoco Principal15 minSetup InicialIniciar projeto Node, instalar dependências (express, multer, mysql2, dotenv) e criar o banco/tabela.20 minConfiguração MulterCriar o arquivo de configuração do diskStorage para renomear arquivos e filtrar extensões.30 minO "C" do CRUD (Upload)Implementar a rota POST que recebe o arquivo e salva o nome/caminho no MySQL.15 minO "R" do CRUD (Listagem)Implementar a rota GET para retornar a lista de arquivos salvos.10 minTestes (Insomnia/Postman)Validar se os arquivos estão chegando na pasta e os dados no banco.
